# SQL_CaseStudy_Flights
This case study shows some practice and data exploration for a few business questions where the concepts of Join, CTE, Window Function, case statement have been implemented.
The Platform which has been used here is MySQL Workbench. There are total 5 tables have been created using CREATE TABLE and insertion of records have been done using INSERT INTO statement.
The ER Diagram also has been kept in this repository which shows the relationships among all the tables.
Thereafter total 10 business questions are present which have been solved using Joins, CTE and Window function.

**Q1. Find the busiest airport by the number of flights take off.**

**Q2. Find the Total number of ticekets sold per airline.**

**Q3. List all the flights operated by 'Indigo' with airport names (Origin and Destination).**

**Q4. For each airport, show the top airline by number of flights departing from there.**

**Q5. For each flight, show time taken in hours and categorize it in short(<2h), Medium(2-5h) or long(>5h).**

**Q6. Show each passanger's first and last flight dates and number of flights.**

**Q7. Find flights with the highest price ticket sold for each route (origin -> destination)**

**Q8. Find the highest spending passenger in each frequent flyer status group.**

**Q9. Find the total revenue and number of ticekts sold by each airline and rank the airlines based in total revenue.**

**Q10. For each passenger, identify their most frequently used airline. If a passenger has multiple airlines with the same highest usage, show all such airlines.**
